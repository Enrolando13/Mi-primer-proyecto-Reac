# 📝 Lista de Tareas con React

## 📌 Descripción
Esta aplicación web permite agregar y eliminar tareas de forma dinámica utilizando React. 
El objetivo es aplicar conceptos básicos como componentes, estado (useState) e interacción con el usuario.

## ⚙️ Tecnologías utilizadas
- React
- Vite
- HTML
- CSS

## 🧩 Componentes principales
- App.jsx: Componente principal que maneja el estado.
- TaskList.jsx: Muestra la lista de tareas.
- TaskItem.jsx: Representa cada tarea individual.

## 🔄 Flujo de la aplicación
1. El usuario escribe una tarea en el input.
2. Presiona el botón "Agregar".
3. La tarea se guarda en el estado (useState).
4. Se renderiza en pantalla.
5. Puede eliminarse con el botón ❌.
